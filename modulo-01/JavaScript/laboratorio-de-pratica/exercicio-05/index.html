Exercício 5

Usando conceitos de objetc e DOM (DocumentObjectModel) do JavaScript, crie um objeto chamado dadosAluno com os atributos: Nome, Curso, Modulo, Professor e declare um valor fictício para cada um dos atributos. Mostre esse objeto em formato de tabela HTML, usando o nome dos atributos como nome dos cabeçalhos das colunas e os valores desses atributos como dados da tabela. Coloque o código JS em um arquivo externo.
