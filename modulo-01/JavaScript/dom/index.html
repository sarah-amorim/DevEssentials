Eventos

  • Os eventos mais comuns
    1. onclick

      2. disparado quando o usuário clica sobre o elemento.

    2. onchange

      3. disparado quando um elemento HTML foi alterado.

    3. onmouseover

      4. disparado quando o usuário passa o mouse “por cima” do elemento.

    4. onmouseout

      5. disparado quando o usuário movimenta o mouse “para fora” do elemento.

    5. onkeydown

      6. disparado quando o usuário pressiona alguma tecla do teclado.

    6. onload

      7. disparado quando o navegador termina de carregar a página.
